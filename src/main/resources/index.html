Руководство по использованию

Перед работой с бд(просмотром и добавлением товаров) ее необходимо создать. В модуле db в пакете org.java_courses запустите класс Main. В этом же пакете в классе CREDS хранятся креды для подключения к бд

В файле ресурсов jdbc_config также необходимо настроить креды для нормальной работы логин сервиса

Чтобы просмотреть товары нужно перейти по url http://localhost:3466/products

Добавить товар можно например так http://localhost:3466/products/add?product=test&amount=12&organization=test01

Пользователи:

Менеджер - логин и пароль = manager

Гость - логин и пароль = guest
